>>Node.js
Node.js是一个事件驱动I/O服务端js环境，基于Google的V8引擎;
可用来部署一些高性能的服务;

>>安装与配置
下载安装nodejs，并将路径(如D:\nodejs\)添加到环境变量;
如果安装成功, 可以通过 cmd->node -v 查看版本;

>>第一个nodejs应用
//server.js
1, 引入需要的模块
var http = require('http');
2, 创建服务器
http.createServer(function(request, response){
	//发送http头部
	response.writeHead(200, {'Content-Type': 'text/plain'});
	//发送响应数据
	response.end('hello, tang bing');
}).listen(8888); //监听8888端口
//终端打印如下信息
console.log('Server running at http://127.0.0.1:8888/');
cmd->node server.js 会输出终端信息
3, 使用浏览器访问
http://127.0.0.1:8888/
显示响应数据;

>>npm
1, npm是同nodejs一起安装的包管理工具;
用户可以上传自己的程序或包到npm服务器，也可以从npm服务器下载别人编写的程序或包;
2, cmd->npm -v  //npm的版本号，可以测试npm是否安装成功
3, 本地安装
cmd->cd到指定安装目录->npm install <ModuleName>
会将包安装在./node_modules下，没有这个目录会自动创建;
可以通过require()来引入本地安装的包;
4, 全局安装
npm install <ModuleName> -g
将安装包放在全局的node/modules目录;
5, package.json
位于模块的目录下，定义了包的属性;
6, 卸载模块
cmd->npm uninstall <ModuleName>
7, 查看所有模块
cmd->npm ls
8, 更新模块
cmd->npm update <ModuleName>
